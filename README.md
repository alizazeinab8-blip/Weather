# 🌤️ Weather App

A modern and responsive weather application built with **React**, **TypeScript**, and **Tailwind CSS**. It allows users to search for any city and view current weather conditions, a 5-day forecast, air quality information, and more.

## ✨ Features

- 🔍 Search weather by city name
- 📍 Get weather using current location
- 🌡️ Current temperature and weather conditions
- 📅 5-Day weather forecast
- 🌅 Sunrise & Sunset times
- 🌫️ Air Quality Index (AQI)
- ⭐ Favorite cities
- 💾 Save last searched city with Local Storage
- 🌙 Light / Dark mode
- 📱 Fully responsive design
- ⚡ Fast API requests with Axios
- 🛡️ Type-safe development with TypeScript

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/alizaeinab8-blip/Weather.git
```

Go to the project

```bash
cd Weather
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
VITE_API_KEY=YOUR_OPENWEATHER_API_KEY
```

Start the development server

```bash
npm run dev
```
